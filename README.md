# Путь к спокойствию

Небольшое статическое веб-приложение для самооценки состояния, работы с
мыслями и планирования полезных действий.

## Локальный запуск

В корне проекта выполните:

```bash
python3 -m http.server 8000
```

После этого откройте <http://localhost:8000>.

## Публикация на GitHub Pages

1. Создайте на GitHub новый пустой репозиторий, например `socialAnxiety`.
   Не добавляйте через сайт README, `.gitignore` или лицензию.
2. Если remote ещё не настроен, подключите репозиторий и отправьте проект:

   ```bash
   git remote add origin https://github.com/ВАШ_ЛОГИН/socialAnxiety.git
   git push -u origin master
   ```

3. На странице репозитория откройте **Settings → Pages**.
4. В разделе **Build and deployment** выберите:
   - **Source:** `Deploy from a branch`
   - **Branch:** `master`
   - **Folder:** `/(root)`
5. Нажмите **Save**. Через несколько минут сайт появится по адресу:

   ```text
   https://ВАШ_ЛОГИН.github.io/socialAnxiety/
   ```

Если репозиторий называется `ВАШ_ЛОГИН.github.io`, адрес сайта будет
`https://ВАШ_ЛОГИН.github.io/`.

## Быстрое обновление проекта

После внесения изменений выполните:

```bash
git add .
git commit -m "Обновить проект"
git push
```

Чтобы задать собственное сообщение коммита одной командой:

```bash
git add . && git commit -m "Кратко опишите изменения" && git push
```

Публикация GitHub Pages обновится автоматически после отправки изменений.

> Папка `docs/` используется для локальных материалов и не отправляется
> в репозиторий.
