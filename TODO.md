# TODO: улучшения приложения для первых 2 недель

Цель первых двух недель: не лечить тревогу силой и не начинать сложные экспозиции, а собрать понятную карту социальной тревожности и общей тревоги: ситуации, прогнозы, защитные поведения, самофокус, избегание, сон/кофеин/движение и первые реалистичные шаги.

## Высокий приоритет

- [ ] Добавить режим "Первые 14 дней"
  - Отдельный маршрут с ежедневными задачами.
  - День 1-2: стартовая оценка и карта проблемы.
  - День 3-7: одна запись социальной ситуации в день.
  - День 8-10: выявление защитных поведений.
  - День 11-14: сбор и оценка лестницы социальных действий.
  - Почему: первые две недели должны быть фазой наблюдения и подготовки, а не резким входом в сложные упражнения.


- [ ] Расширить дневник ситуаций
  - Добавить поля:
    - где было;
    - кто был рядом;
    - тревожный прогноз в процентах;
    - использованное защитное поведение;
    - фокус внимания: на себе / наружу;
    - сколько времени было пережевывание после ситуации.
  - Почему: для социальной тревожности важны не только событие и факт, но и самофокус, защитные действия и постсобытийный анализ.

- [ ] Добавить чеклист защитных поведений
  - Варианты:
    - смотрел в телефон;
    - говорил тише обычного;
    - избегал взгляда;
    - заранее репетировал фразу;
    - молчал, чтобы не ошибиться;
    - стоял у выхода;
    - чрезмерно объяснял себя;
    - просил подтверждения, что все нормально;
    - быстро завершал разговор;
    - использовал алкоголь, никотин или кофеин как опору.
  - Оставить возможность добавить свой вариант.
  - Почему: сначала нужно научиться замечать защитные поведения, и только потом постепенно их уменьшать.

- [ ] Переработать экран "Действия" под фазу подготовки
  - Для первых двух недель показывать цель: собрать 10-15 социальных действий и оценить тревогу.
  - Добавить прогресс: "собрано X из 15".
  - Подсвечивать, есть ли действия разных уровней: 1-2/10, 3-4/10, 5-7/10, 8-10/10.
  - Не подталкивать к выполнению действий 3-4/10 до окончания фазы подготовки.
  - Почему: корректная лестница страха нужна до начала регулярных экспозиций.

## Средний приоритет

- [ ] Добавить "карту круга тревоги"
  - Схема: ситуация -> тревожный прогноз -> симптомы -> защитное поведение -> краткое облегчение -> усиление страха.
  - Заполнять ее из дневника и чеклиста.
  - Почему: пользователю легче работать с тревогой, когда он видит механизм, а не отдельные симптомы.

- [ ] Добавить обзор 7-го и 14-го дня
  - Вопросы:
    - какие ситуации повторялись чаще всего;
    - чего я чаще всего боялся;
    - какие защитные поведения встречались чаще всего;
    - что фактически происходило;
    - какой самый маленький следующий шаг выглядит реалистичным.
  - Почему: дневник должен превращаться в выводы, иначе записи остаются просто архивом.

- [ ] Добавить трекер постсобытийного пережевывания
  - Отметки времени: 0-5 минут, 5-20 минут, 20-60 минут, больше часа.
  - Кнопка "закрыть разбор" с тремя вопросами:
    - что произошло фактически;
    - что я сделал несмотря на тревогу;
    - что попробую в следующий раз.
  - Почему: длительный анализ разговоров после события поддерживает социальную тревогу.

- [ ] Добавить короткие карточки психообразования
  - Темы:
    - цель не в нулевой тревоге;
    - почему избегание усиливает страх;
    - что такое защитное поведение;
    - почему факты важнее догадок;
    - почему дыхание не должно становиться ритуалом перед каждым разговором.
  - Формат: короткие карточки, без длинных статей.
  - Почему: первые две недели должны объяснять модель тревоги простым языком.

- [ ] Добавить мягкую защиту от перегруза
  - Если пользователь планирует слишком много задач на день, показывать напоминание: для первых двух недель достаточно одной записи и одного наблюдения.
  - Не использовать жесткие streak-наказания.
  - Почему: тревожный пользователь может превратить лечение в перфекционистский проект.

## Низкий приоритет, но полезно

- [ ] Добавить экспорт данных
  - Форматы: JSON и простой текст.
  - Включать дневник, шкалы, список действий, защитные поведения и обзоры недели.
  - Почему: данные первых двух недель можно показать психологу или врачу.

- [ ] Добавить управление приватностью
  - Объяснить, что данные хранятся локально в браузере.
  - Добавить кнопку удаления всех данных.
  - Добавить резервную копию / восстановление.
  - Почему: страх раскрытия личной информации может мешать честно вести записи.

- [ ] Добавить блок "когда нужна помощь специалиста"
  - Указать красные флаги:
    - суицидальные мысли;
    - выраженная депрессия;
    - резкое ухудшение сна или работы;
    - алкоголь или вещества для общения;
    - самоповреждение;
    - тревога почти полностью блокирует жизнь;
    - панические симптомы или частые кризисы.
  - Тон должен быть спокойным, без запугивания.
  - Почему: приложение не должно заменять врача или психотерапевта в ситуациях высокого риска.

## Чего не добавлять в первые 2 недели

- [ ] Не добавлять агрессивные челленджи.
- [ ] Не добавлять публичные достижения или социальные функции.
- [ ] Не делать streak-наказания.
- [ ] Не заставлять каждый день выполнять социальную экспозицию.
- [ ] Не делать дыхание обязательным ритуалом перед разговором.
- [ ] Не перегружать пользователя длинной теорией.
